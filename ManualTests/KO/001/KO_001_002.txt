Id -> KO_001_002
Name -> Adding a few different items to cart.

Brief description -> This test verifies the functionality of adding a few different items to the shopping cart.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Reject cookie-files.
	7. Button "Přidat do košíku" on webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".

Test steps ->
	1. Click the button "Přidat do košíku".
	2. On the opened webpage click button "Zpět do obchodu".
	3. Open a new webpage using URL "https://www.lokni-shop.cz/prislusenstvi-k-lahvim-2/obal-na-lahev-seahorse/".
	4. 	TEST - webpage "Obal na láhev Seahorse" is opened.
	5. Click the button "Přidat do košíku".
	6. On the opened webpage click button "Přejít do košíku".

Input data ->
	1. Product's URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	2. Product's URL "https://www.lokni-shop.cz/prislusenstvi-k-lahvim-2/obal-na-lahev-seahorse/".

Expected result ->
	1. Two items are added to the shopping cart: "Nerezová termoláhev Proaqua modrá 500 ml"(1pc), "Obal na láhev Seahorse"(1pc).

Post-conditions ->
	1. Remove all the items from the shopping cart.
	2. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1h
