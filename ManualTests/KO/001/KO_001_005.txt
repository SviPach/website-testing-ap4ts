Id -> KO_001_005
Name -> Attempt to add an invalid quantity of item to the cart.

Brief description -> This test verifies if it is possible to add an invalid quantity of item to the shopping cart.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Reject cookie-files.
	7. Button "Přidat do košíku" on webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".
	8. Product quantity input field on webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".

Test steps ->
	1. Enter the value "999" in the product quantity input field.
	2. Click button "Přidat do košíku".

Input data ->
	1. Product's URL "https://www.lokni-shop.cz/domaci-filtrace/domaci-filtrace-lokni-home-ultimate/".
	2. Text "999".

Expected result ->
	1. The message "Tento produkt má omezený počet kusů, které je možno zakoupit. Dostupných je pouze X položek." must be displayed.

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1f
