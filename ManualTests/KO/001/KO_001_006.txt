Id -> KO_001_006
Name -> Adding more than 9999pc of item to the cart.

Brief description -> This test verifies if it is possible to add more than 9999pc of the same item to the shopping cart.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Reject cookie-files.
	7. Product quantity input field on webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".

Test steps ->
	1. Ener value "9999" in the product quantity input field.

Input data ->
	1. Product's URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	2. Text "9999".

Expected result ->
	1. The message "Není možné zakoupit více než 9999 ks." must be displayed.

Post-conditions ->
	1. Close the website.
	2. Odstraníme všechny produkty z koše.

TC version -> 0.0.0.0.0.0.0.0.1d
