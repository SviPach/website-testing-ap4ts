Id -> KO_002_002
Name -> Changing the quantity of the item in the cart.

Brief description -> This test verifies the functionality of changing the item quantity in the shopping cart.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Reject cookie-files.
	7. Button "Přidat do košíku" on webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".

Test steps ->
	1. Click button "Přidat do košíku".
	2. On the opened webpage click button "Přejít do košíku".
	3. Click the "+" button once to the right of the product "Nerezová termoláhev Proaqua modrá 500 ml".

Input data ->
	1. URL produktu "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".

Expected result ->
	1. Two items are added to the shopping cart: "Nerezová termoláhev Proaqua modrá 500 ml" (2pc).

Post-conditions ->
	1. Remove all the items from the shopping cart.
	2. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1c
