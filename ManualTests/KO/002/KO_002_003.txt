Id -> KO_002_003
Name -> Removing the item from the cart - fast access.

Brief description -> This test verifies the functionality of removing the item from the shopping cart using the fast access window.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Reject cookie-files.
	7. Button "Přidat do košíku" on webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".
	8. Shopping cart icon in the right top corner.

Test steps ->
	1. Click button "Přidat do košíku".
	2. On the opened webpage click button "Zpět do obchodu".
	3. Point the mouse cursor on the shopping cart icon in the right top corner.
	4. In the drop-down window click on the shopping cart icon to the right of the product "Nerezová termoláhev Proaqua modrá 500 ml".

Input data ->
	1. Product's URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".

Expected result ->
	1. The message "Položka košíku byla odebrána." must be displayed.
	2. Shopping cart is empty with the messgae "Košík je prázdný. Naplňte ho radostí!" in the drop-down window.

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1b
