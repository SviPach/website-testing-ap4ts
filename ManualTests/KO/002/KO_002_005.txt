Id -> KO_002_005
Name -> Changing the quantity of the item in the cart to the unavailable amount.

Brief description -> This test verifies the functionality of changing the product quantity in the shopping cart to the unavailable amount.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Reject cookie-files.
	7. Button "Přidat do košíku" on webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".

Test steps ->
	1. Click button "Přidat do košíku".
	2. On the opened webpage click button "Přejít do košíku".
	3. Enter value "999" in the product quantity input field.
	4. Click "Enter" on the keyboard.

Input data ->
	1. Product's URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	2. Text "999".

Expected result ->
	1. The message "Tento produkt má omezený počet kusů, které je možno zakoupit. Dostupných je pouze X položek. Množství nebylo změněno." must be displayed.
	2. Quantity of the product "Nerezová termoláhev Proaqua modrá 500 ml" is not changed.

Post-conditions ->
	1. Remove all the items from the shopping cart.
	2. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1d
