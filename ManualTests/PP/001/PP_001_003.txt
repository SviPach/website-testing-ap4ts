Id -> PP_001_003
Name -> Switching the images in a full screen mode.

Brief description -> This test verifies the functionality of the image switching in a full screen mode.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Reject cookie-files.
	7. Main image on the webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".

Test steps ->
	1. Click the main image.
	2. 	TEST - Image is opened in full screen mode.
	3. Click the right side of the image.

Input data ->
	1. Product's URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".

Expected result ->
	1. Image should be switched to the next one.

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1e
