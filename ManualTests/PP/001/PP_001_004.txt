Id -> PP_001_004
Name -> Switching the images without a full screen mode.

Brief description -> This test verifies the functionality of the image switching without a full screen mode.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Reject cookie-files.
	7. Image panel on the webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".
	8. Second image in the image panel on the webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".

Test steps ->
	1. Click the image right to the first one in the image panel.

Input data ->
	1. Product's URL "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".

Expected result ->
	1. Po kliknutí na obrázek se musí zobrazit jako hlavni.

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1d
