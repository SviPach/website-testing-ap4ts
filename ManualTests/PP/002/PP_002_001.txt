Id -> PP_002_001
Name -> Moving to the product's ratings.

Brief description -> This test verifies the functionality of quick scrolling to the product's ratings panel.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz/domaci-filtrace/domaci-filtrace-lokni-home-ultimate/".
	6. Reject cookie-files.
	7. Button "# HODNOCENÍ" on the webpage "Domácí filtrace Lokni ULTIMATE - LOKNI-SHOP".

Test steps ->
	1. Click the "# HODNOCENÍ" button under the product's name, where # - amount of ratings.

Input data ->
	1. Product's URL "https://www.lokni-shop.cz/domaci-filtrace/domaci-filtrace-lokni-home-ultimate/".

Expected result ->
	1. Webpage has to move us down to the "Hodnocení (#)" panel.

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1d
