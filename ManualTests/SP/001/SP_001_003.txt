Id -> SP_001_003
Name -> Sorting "Domácí filtrace - LOKNI-SHOP". By alphabet.

Brief description -> This test verifies the functionality of the "ABECEDNĚ" sort button on the webpage "Domácí filtrace - LOKNI-SHOP".

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. Reject cookie-files.
	7. Button "Domácí filtrace" on the webpage "Jedna láhev stačí - LOKNI-SHOP" in the "Kategorie" panel.

Test steps ->
	1. Click the "Domácí filtrace" button in the "Kategorie" panel.
	2. 	TEST - webpage "Domácí filtrace - LOKNI-SHOP" is opened.
	3. In the line with the sort buttons click the "ABECEDNĚ" button.

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".

Expected result ->
	1. Products are sorted by alphabet (A - Ž).

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.4k
