Id -> TL_001_001
Name -> Opening the homepage.

Brief description -> This test verifies the functionality of the homepage's button.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. Reject cookie-files.
	7. Button "lokni" on the icon's place on the webpage "Jedna láhev stačí - LOKNI-SHOP".

Test steps ->
	1. Click on the website's icon "lokni".

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".

Expected result ->
	1. Website should redirect us to the webpage  "Jedna láhev stačí - LOKNI-SHOP" (https://www.lokni-shop.cz).

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.4j
