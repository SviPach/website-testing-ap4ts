Id -> TL_001_004
Name -> Opening the "Láhve LOKNI - LOKNI-SHOP" webpage using a top panel.

Brief description -> This test verifies the functionality of the "LÁHVE LOKNI" button on the top panel.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. Reject cookie-files.
	7. Button "LÁHVE LOKNI" on the top panel on the webpage "Jedna láhev stačí - LOKNI-SHOP".

Test steps ->
	1. Click the "LÁHVE LOKNI" button on the top panel.

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".

Expected result ->
	1. Website should redirect us to the webpage "Láhve LOKNI - LOKNI-SHOP" (https://www.lokni-shop.cz/lahve/).

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.2h
