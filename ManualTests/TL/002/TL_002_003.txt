Id -> TL_002_003
Name -> Collapsing the "Láhve LOKNI" button on the "Kategorie" panel.

Brief description -> This test verifies the functionality of the collapsing feature of the "Láhve LOKNI" button on the "Kategorie" panel.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. Reject cookie-files.
	7. Button "Kuchyňské baterie" on the webpage "Jedna láhev stačí - LOKNI-SHOP".

Test steps ->
	1. On the left from the "Láhve LOKNI" button on the "Kategorie" panel click the down arrow.

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".

Expected result ->
	1. Button "Skleněné láhve LOKNI" should be collapsed.
	2. Button "Nerezové láhve LOKNI" should be collapsed.
	3. Button "Zachraň lahev" should be collapsed.

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1f
