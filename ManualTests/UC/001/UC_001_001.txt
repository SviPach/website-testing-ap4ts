Id -> UC_001_001
Název -> Přihlášení bez e-mailu.

Stručný popis -> Testovací případ testující funkcionalitu tlačítka "PŘIHLÁŠENÍ" bez zadání e-mailu.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. Musí být zapnutý WI-FI.
	3. Myš, klávesnice.
	4. Pokud web už byl otevřen - musíme ho zavřít.
	5. Web otevřen pomoci odkazu "https://www.lokni-shop.cz".
	6. Pokud jsme přihlášený - musíme se odhlásit.
	7. Odmítněme cookies.
	8. Rozbalovací tlačítko "PŘIHLÁŠENÍ" na stránce "Jedna láhev stačí - LOKNI-SHOP".

Testovací kroky ->
	1. Klikneme na rozbalovací tlačítko "PŘIHLÁŠENÍ" nahoře doprava.
	2. 	TEST - Tlačítko "PŘIHLÁŠENÍ" se rozbalilo.
	3. Klikneme v rozbaleném okně na tlačítko "Přihlásit se".

Vstupní data ->
	1. URL domové stránky "https://www.lokni-shop.cz".

Očekávaný výsledek ->
	1. Musí se zobrazit malé okno s textem "Please fill out this field" pod polem "E-MAIL".

Post-conditions ->
	1. Zavřeme web.

Verze TC -> 0.0.0.0.0.0.0.0.3j
