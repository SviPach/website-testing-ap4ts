Id -> UC_003_005
Name -> Adding a new delivery address.

Brief description -> This test verifies the functionality of the delivery address adding.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. If we are logged in - have to sign out.
	7. Reject cookie-files.
	8. Drop-down button "PŘIHLÁŠENÍ" on the webpage "Jedna láhev stačí - LOKNI-SHOP".

Test steps ->
	1. Click the drop-down "PŘIHLÁŠENÍ" button in the top right corner.
	2. 	TEST - Button "PŘIHLÁŠENÍ" is expanded.
	3. Enter "mimoli1034@noomlocs.com" in the input field "E-MAIL" in the expanded window of the "PŘIHLÁŠENÍ" button.
	4. Enter "123456789" in the input field "HESLO" in the expanded window of the "PŘIHLÁŠENÍ" button.
	5. On the opened window click the button "Přihlásit se".
	6. 	TEST - sign in is successful.
	7. Click the "MŮJ ÚČET" button in the top right corner.
	8. Click the "Moje osobní údaje" button on the right panel.
	9. Go to the bottom and click the "Přidat novou adresu" button in the "Doručovací adresy" section.
	10. Enter value "Marcel Orel" in the input field "JMÉNO A PŘIJMENÍ".
	11. Enter value "Zlínská 1050" in the input field "ULICE".
	12. Enter value "Zlín" in the input field "MĚSTO".
	13. Enter value "76314" in the input field "PSČ".
	14. In the field "ZEMĚ" choose "Slovensko".
	15. Click the "Odeslat" button.

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".
	2. E-mail "mimoli1034@noomlocs.com".
	3. Password "123456789".
	4. Text "Marcel Orel".
	5. Text "Zlínská 1050".
	6. Text "Zlín".
	7. Text "76314".

Expected result ->
	1. The message should be shown "Záznam byl v pořádku uložen.".
	2. New delivery address was added on the second position.
	3. "Jméno a příjmení" of the new delivery address is "Marcel Orel".
	4. "Ulice" of the new delivery address is "Zlínská 1050".
	5. "Město" of the new delivery address is "Zlín".
	6. "PSČ" of the new delivery address is "76314".
	7. "Země" of the new delivery address is "Slovensko".

Post-conditions ->
	1. Close the website.
