Id -> UC_003_006
Name -> Deleting the delivery address.

Brief description -> This test verifies the functionality of the delivery address deleting.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. Successful completing of the UC_003_005 test.
	7. Reject cookie-files.
	8. Drop-down button "PŘIHLÁŠENÍ" on the webpage "Jedna láhev stačí - LOKNI-SHOP".

Test steps ->
	1. Click the drop-down "PŘIHLÁŠENÍ" button in the top right corner.
	2. 	TEST - Button "PŘIHLÁŠENÍ" is expanded.
	3. Enter "mimoli1034@noomlocs.com" in the input field "E-MAIL" in the expanded window of the "PŘIHLÁŠENÍ" button.
	4. Enter "123456789" in the input field "HESLO" in the expanded window of the "PŘIHLÁŠENÍ" button.
	5. On the opened window click the button "Přihlásit se".
	6. 	TEST - sign in is successful.
	7. Click the "MŮJ ÚČET" button in the top right corner.
	8. Click the "Moje osobní údaje" button on the right panel.
	9. Go to the bottom and click the "Smazat" button in the "Smazat" column in the "Doručovací adresy" section next to the address we created in the UC_003_005 test.
	10. In the displayed alert-window click the "OK" button.

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".
	2. E-mail "mimoli1034@noomlocs.com".
	3. Password "123456789".

Expected result ->
	1. The message "Záznam byl úspěšně odstraněn." should be displayed.
	2. The delivery address, created during UC_003_005 test should be deleted.

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1g
