Id -> VY_001_001
Name -> Searching the "Proaqua" bottles.

Brief description -> This test verifies that the product search bar correctly handles cases when products are found.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. Reject cookie-files.
	7. Search bar in the top of the webpage "Jedna láhev stačí - LOKNI-SHOP".

Test steps ->
	1. Click on the input field of the search bar in the top of the webpage.
	2. 	TEST - text input mode is active.
	3. Enter the text "Proaqua".
	4. Click the "Enter" button on the keyboard.

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".
	2. Text "Proaqua".

Expected result ->
	1. Website should redirect us to the webpage "Vyhledávání - LOKNI-SHOP" (https://www.lokni-shop.cz/vyhledavani/?string=Proaqua).
	2. On the webpage "Vyhledávání - LOKNI-SHOP" in the "Výsledky vyhledávání "Proaqua"" section the product should be showed: "Nerezová termoláhev Proaqua stříbrná 500 ml".
	3. On the webpage "Vyhledávání - LOKNI-SHOP" in the "Výsledky vyhledávání "Proaqua"" section the product should be showed: "Nerezová termoláhev Proaqua modrá 500 ml".

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1g
