Id -> VY_001_002
Name -> Searching the "Proaqua" bottles - fast access.

Brief description -> This test verifies that the product search bar correctly handles cases when products are found and are shown on the fast access window.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. Reject cookie-files.
	7. Search bar in the top of the webpage "Jedna láhev stačí - LOKNI-SHOP".

Test steps ->
	1. Click the input field of the search bar in the top of the webpage.
	2. 	TEST - text input mode is active.
	3. Enter the text "Proaqua".
	4. 	TEST - On the fast access window 2 products are shown.
	5. Click the first bottle "Nerezová termoláhev Proaqua modrá 500 ml" from the fast access window.

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".
	2. Text "Proaqua".

Expected result ->
	1. Website should redirect us to the webpage "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP" (https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/).

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1с
