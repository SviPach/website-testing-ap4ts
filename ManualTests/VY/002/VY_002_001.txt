Id -> VY_002_001
Name -> Searching the "Proaquap" bottles.

Brief description -> This test verifies that the product search bar correctly handles cases when no products are found.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. WI-FI has to be turned on.
	3. Mouse, keyboard.
	4. If the website is opened already - have to close it.
	5. Website is opened using URL "https://www.lokni-shop.cz".
	6. Reject cookie-files.
	7. Search bar in the top of the webpage "Jedna láhev stačí - LOKNI-SHOP".

Test steps ->
	1. Click the input field of the search bar in the top of the webpage.
	2. 	TEST - text input mode is active.
	3. Enter the text "Proaquap".
	4. Click the "Enter" button on the keyboard.

Input data ->
	1. Homepage's URL "https://www.lokni-shop.cz".
	2. Text "Proaquap".

Expected result ->
	1. Website should redirect us to the webpage "Vyhledávání - LOKNI-SHOP" (https://www.lokni-shop.cz/vyhledavani/?string=Proaquap).
	2. On the webpage "Vyhledávání - LOKNI-SHOP" in the "Výsledky vyhledávání "Proaquap"" section the text should be showed: "Produkty nebyly nalezeny".

Post-conditions ->
	1. Close the website.

TC version -> 0.0.0.0.0.0.0.0.1f
