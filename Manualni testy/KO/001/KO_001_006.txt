Id -> KO_001_006
Název -> Přidání více 9999ks produktu do košíku.

Stručný popis -> Testovací případ testující funkcionalitu přidáni více něž 9999ks stejného produktu do nákupního košíku.

Pre-conditions -> 
	1. Windows 11 - [Version 10.0.22631.22631].
	2. Musí být zapnutý WI-FI.
	3. Myš, klávesnice.
	4. Pokud web už byl otevřen - musíme ho zavřít.
	5. Web otevřen pomoci odkazu "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	6. Odmítněme cookies.
	7. Textové pole pro zadání množství produktu na stránce "Nerezová termoláhev Proaqua modrá 500 ml - LOKNI-SHOP".

Testovací kroky ->
	1. Zadáme hodnotu "9999" v textovém poli pro zadání množství produktu.

Vstupní data ->
	1. URL produktu "https://www.lokni-shop.cz/zachran-lahev/nerezova-lahev-proaqua-modra/".
	2. Text "9999".

Očekávaný výsledek ->
	1. Musí se nám zobrazit označení "Není možné zakoupit více než 9999 ks.".
	2. Odstraníme všechny produkty z koše.

Post-conditions ->
	1. Zavřeme web.

Verze TC -> 0.0.0.0.0.0.0.0.1d
